Бэкенд для системы менеджмента проектов

Необходимо написать REST API, предоставляющий функционал для
управления проектами, заведения задач и т.д.

-- Методы API --
- Пользователи:
	- Регистрация пользователя - Готово
	- Авторизация пользователя - Готово
- Проекты:
	- Создание проекта - Готово
	- Получение списка проектов - Готово
- Задачи:
	- Заведение задачи в рамках проекта Готово
	- Поиск задач с фильтрацией по полям - не понял как реализовать, сделал поиск по имени
	- Получение задачи Готово(реализовано: во время создания задачи в параметре user_assigment_task = "User")
	- Добавление комментария к задаче Готово
	- Изменения статуса задачи Готово

-- Модели --
Модель Проекта:
	- Название проекта

Модель пользователя:
	- юзернейм
	- пароль

Модель задачи:
	- Название задачи
	- Описание задачи
	- статус задачи
	- проект, в рамках которого заведена задача
	- дата заведения задачи
	- дата закрытия задачи
	- Пользователь, который завел задачу
	- Пользователь, на которого завели задач
	- список файлов - (загрузку файлов сделал, отображение в бд не сделал )
	- список комментариев

Модель комментария:
	- Текст
	- Автор комментария

-- Требования к проекту --
	- пароли должны храниться в базе в зашифрованном виде - Готово (17.01 20:37)
	- обращаться к методам в разделах "Проекты" и "Задачи" могут только зарегистрированные пользователи
	- БД - Postgres (подключил онлайн через ElephantSql.com, не мог создать на своей машине, выходила ошибка)
	- проект должен разворачиваться в докере (tap1x/testtask:image)

Необходимые библиотеки:
- flask
- marshmallow
- sqlalchemy
